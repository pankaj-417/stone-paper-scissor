Stone Paper Scissors Game

A simple and fun browser-based Stone-Paper-Scissors game built while learning from Apna College.
This project helped me practice JavaScript basics, DOM manipulation, and event handling.

✨ Features

Play against the computer with random moves

Clean and interactive UI

Real-time score updates

Beginner-friendly project for practicing JavaScript

🛠️ Tech Stack

HTML5 for structure

CSS3 for styling

JavaScript (ES6) for logic and interactivity

🚀 How to Play

Choose Stone 🪨, Paper 📄, or Scissors ✂️

Computer makes a random choice

Winner is decided instantly & score updates

📚 Learning Outcome

Strengthened understanding of functions & conditionals in JavaScript

Learned how to manipulate DOM dynamically

Gained hands-on practice in building interactive web apps
